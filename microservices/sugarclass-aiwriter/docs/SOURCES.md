## Source Details

### **How Sources Were Selected**

**Criteria**:
1. ✅ **Free Access** - No paywalls (removed 4 paywalled sources)
2. ✅ **Full Articles** - Complete text via RSS or extractable HTML
3. ✅ **RSS Feed** - Standard RSS/Atom format
4. ✅ **Age-Appropriate** - Content suitable for target age groups
5. ✅ **Reliable** - Established, reputable publishers
6. ✅ **Diverse** - Covers all 12 categories
7. ✅ **Active** - Regular updates (daily/weekly)
8. ✅ **Images** - Visual content available

**Rejected Sources**:
- Economist (paywall)
- New York Times (paywall)
- Washington Post (paywall)
- Wall Street Journal (paywall)
- The Atlantic (paywall)
- Harvard Business Review (paywall)
- Rolling Stone (partial paywall)

## Adding New Sources

### **Step 1: Verify RSS Feed**

```powershell
# Test RSS feed validity
python -c "import feedparser; d = feedparser.parse('YOUR_RSS_URL'); print(f'Title: {d.feed.title}'); print(f'Entries: {len(d.entries)}')"
```

### **Step 2: Add to simple_collector.py**

```python
# simple_collector.py
SOURCES = [
    {
        'name': 'Your Source Name',
        'url': 'https://example.com/rss',
        'category': 'General News',  # See CATEGORIES.md for list
        'age_groups': ['14-16', '17+']  # Target age groups
    },
    # ... existing sources
]
```

### **Step 3: Test Extraction**

```powershell
# Test single source
python -c "
from simple_collector import fetch_full_article
import requests

# Get first article from feed
import feedparser
d = feedparser.parse('YOUR_RSS_URL')
url = d.entries[0].link

# Test extraction
article_data = fetch_full_article(url)
print(f'Title: {article_data.get(\"title\")}')
print(f'Text length: {len(article_data.get(\"text\", \"\"))} chars')
print(f'Has image: {bool(article_data.get(\"image_url\"))}')
"
```

### **Step 4: Collect Articles**

```powershell
python simple_collector.py
```

### **Step 5: Verify in Database**

```powershell
python -c "
import database
db = database.get_db()
count = db.execute('SELECT COUNT(*) FROM articles WHERE source_name = ?', ('Your Source Name',)).fetchone()[0]
print(f'Articles from Your Source Name: {count}')
"
```
